Attribute Selectors:-

Attribute selectors in CSS allow us to select elements based on the presence or value of an attribute.

[attr] - Represents elements with an attribute name of attr regardless of its value.

attr[lang] - Represents elements with an attribute name of attr regardless of its value.

[attr = value] - Represents elements with an attribute name of attr whose value is exactly value.

[attr ~= value] - Represents elements with an attribute name of attr whose value contains the word exactly value.

[attr^=value] - Represents elements with an attribute name of attr whose value is prefixed (preceded) by value.

[attr$=value] - Represents elements with an attribute name of attr whose value is suffixed (followed) by value.
