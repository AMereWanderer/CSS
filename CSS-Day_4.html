Pseudo-Class Selectors:-

A CSS pseudo-class is a keyword added to a selector that specifies a special state of the selected elements

Tree-structural pseudo-classes:- These pseudo-classes relate to the location of an element within the document tree.

:first-child -
This CSS pseudo-class represents the first element & its first occurrence among a group of sibling elements.
